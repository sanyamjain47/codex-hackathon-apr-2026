# Viewer

This package is the local web UI scaffold.

## Current State

The viewer is a Vite React app with a placeholder screen. It is not wired to an MCP/session backend yet.

## Development

```bash
npm run dev --workspace @git-diff-viewer/viewer
```

The dev server binds to `127.0.0.1`.
